# Jack Randolph Public Lab

Public-facing portfolio and personal code cheatsheet inventory for Jack Randolph.

## Purpose

- A branded home for software, media, and project work.
- A growing inventory of reusable CSS, JavaScript, React, and prompt patterns.
- A GitHub-ready source that can later be connected to Squarespace or folded into the broader public rebrand.

## Development

```bash
npm install
npm run dev
```

## Brand Notes

The first pass follows `JR Rebrand Guide.pdf`: charcoal `#363436`, hot red `#ec3724`, off-white `#f4f4f5`, bold rounded typography, friendly rounded geometry, waves, skull/star accents, and the Video / Design / Software language.
