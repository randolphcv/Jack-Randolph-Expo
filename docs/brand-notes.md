# Brand Notes

Source: `JR Rebrand Guide.pdf`.

- Core colors: `#363436`, `#ec3724`, `#f4f4f5`.
- Typography direction: SF Pro Rounded logotype feel, Neue Haas Grotesk Display Pro secondary feel.
- Visual direction: bolder and friendlier, no right angles, pirate motif as usual.
- Motifs: rounded blocks, wave dividers, subtle skull pattern, red sparkle accent, black-and-white photography with selective red emphasis.
- Language fragments from the guide: `Video.`, `Design.`, `Software.`, `Making Lists.`, `Describing things.`, `Being obnoxious.`
